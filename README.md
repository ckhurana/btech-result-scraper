## Docker Setup

This project is now Dockerized for easy deployment and local development.

### Running with Docker Compose

1. **Build and Start:**
   ```bash
   docker compose up -d --build
   ```

2. **Stop:**
   ```bash
   docker compose down
   ```

The application will be available at its mapped port (see root README for details).
